KOMPONEN DASAR PENYUSUN WEBSITE

  1. HTML

    2. HTML adalah kependekan dari Hypertext Markup Language yang merupakan sebuah bahasa markup. HTML adalah kode untuk membuat struktur halaman suatu website yang menarik, saling terhubung satu dengan yang lainnya, dan yang pasti dapat diakses melalui internet.
    Awalnya html ditemukan oleh Tim Berners-Lee pada tahun 1991. HTML adalah solusi untuk membantu ilmuwan dalam mengakses dokumen, namun kini html semakin berkembang pesat di dunia pemrograman web. Bagi pemula yang ingin memulai belajar pemrograman web dan bingung belajar bahasa markup apa, HTML adalah jawabannya. HTML dianggap mudah dipahami oleh orang awam karena tag dasar html mudah dibaca dan diimplementasikan. HTML biasanya ditempatkan atau disimpan pada sebuah file bernama file HTML. Isi dari file tersebut yaitu kombinasi simbol dan juga teks.

  2. CSS

    3. CSS adalah singkatan dari Cascading Style Sheets. Ini adalah bahasa pemrograman yang digunakan untuk mendesain tampilan halaman web. Dengan CSS, Anda dapat mengontrol tata letak, warna, font, dan berbagai atribut visual lainnya dari elemen-elemen HTML di halaman web Anda. CSS memisahkan presentasi dari struktur konten HTML, memungkinkan pengembang untuk membuat tampilan yang konsisten dan mudah diatur ulang tanpa harus mengubah struktur dasar HTML.

  3. JAVASCRIPT

    4. JavaScript adalah bahasa pemrograman tingkat tinggi yang sering digunakan untuk mengembangkan aplikasi web interaktif. Dibuat pertama kali oleh Netscape pada tahun 1995, JavaScript telah berkembang menjadi salah satu bahasa pemrograman paling populer di dunia. JavaScript digunakan untuk memberikan interaktivitas pada halaman web, seperti membuat efek animasi, mengelola formulir, menanggapi input pengguna, dan berkomunikasi dengan server untuk mengambil atau mengirim data tanpa memuat ulang halaman secara penuh (biasanya menggunakan teknik yang dikenal sebagai AJAX). Selain itu, JavaScript juga digunakan untuk mengembangkan berbagai jenis aplikasi web modern, seperti aplikasi berbasis klien (client-side), aplikasi berbasis server (server-side), game web, aplikasi mobile, dan bahkan aplikasi desktop melalui kerangka kerja seperti Electron.
